# ks-media

## 광신방송예술고등학교 필름사진 전시전 홈페이지

### 소개
광신방송예술고등학교 사진동아리의 필름사진 전시전을 위해 만들어진 홈페이지 입니다.
복잡한 시스템을 사용하지 않고 순수 HTML로 만들어졌습니다.
Google Firebase를 사용하여 배포 했습니다.
아래의 링크를 통해 firebase로 배포된 사이트를 방문할 수 있습니다.
